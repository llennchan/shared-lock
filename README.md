# shared-lock

## 介绍
分布式共享锁，暂时支持 redis，支持多种模式
1. try/finally 原始模式
2. callback 回调模式，类似 JdbcTemplate execute
3. AOP 切面注解模式

支持 enable 引用 和 starter 的开箱即用方式。

## 软件架构

依赖 spring + spring-data-redis
